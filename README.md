# Carb0n Cyber Manager - CCM

Sistema para gerenciamento de cyber café / lan house.

---

## Dependencias
### Python 3.13.5
### Flask
### Pillow
### SQLAlchemy

---

## Como rodar o servidor

1. Instale dependências:

```bash
pip install -r docs/requirements.txt
```
---

```bash
cd servidor
python server.py
```

## Como iniciar o painel
```bash
cd painel
python main.py
```
